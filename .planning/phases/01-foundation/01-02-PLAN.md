| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-01 | Information Disclosure | VITE_API_URL in bundle | accept | Points to localhost only in dev build; production URL not set in Phase 1; no secrets exposed in frontend bundle |
| T-02-02 | Tampering | Seed data in production | mitigate | seed.py guards: if ENVIRONMENT == "production": return immediately; docker-compose command runs seed only in dev |

| T-02-04 | Spoofing | CORS in CI test environment | accept | CI tests run directly against ASGI transport or localhost; CORS not exercised in unit tests |
| T-02-SC | Tampering | npm install from package.json | mitigate | All frontend packages from official registries (Package Legitimacy Audit in RESEARCH.md); no [ASSUMED]/[SUS] npm packages; frontend CI runs npm ci (lockfile-pinned) |
