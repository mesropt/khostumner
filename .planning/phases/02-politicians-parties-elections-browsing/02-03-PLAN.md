## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → GET /api/parties/{slug} | Slug path param is untrusted; SQLAlchemy parameterized query prevents injection |
| FastAPI → PostgreSQL | Promise join query must always filter moderation_status=approved |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-09 | Tampering | GET /api/parties/{slug} path param | mitigate | SQLAlchemy: select(Party).where(Party.slug == slug) — ORM parameterized; no string interpolation into SQL |
| T-02-10 | Information Disclosure | GET /api/parties/{slug}/promises | mitigate | Promise query filters moderation_status == ModerationStatus.approved; pending/rejected never returned |
| T-02-11 | Denial of Service | GET /api/parties/{slug}/politicians | accept | Returns all members (not paginated) — party membership count is small (< 100 in v1 seed data); acceptable |
</threat_model>
