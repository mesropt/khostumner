## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, badge internal padding, inline icon-text gap |
| sm | 8px | Card padding internal elements, filter dropdown gap |
| md | 16px | Default element spacing, card body padding, page horizontal padding |
| lg | 24px | Section padding, gap between filter row and results grid |

### 1. Shared Layout (`Layout.tsx`) — wraps all pages

```
┌─────────────────────────────────────────────────────┐
│ HEADER  h=56px  bg=white  border-b=zinc-200         │
│  [Խոստումներ logo]  [Կատարված]  [Չկատարված]          │

└─────────────────────────────────────────────────────┘
```

- Header: `position: sticky top-0 z-10`. No hamburger menu (deferred to Phase 3, per D-03).

- Nav rendered in horizontal row with `gap-24px`. No wrapping on desktop.
- Mobile: nav links wrap naturally; no hamburger (D-03).

## Interaction Contracts

### Filter Dropdowns (PersonsPage)

- Party Select: label "Կուսակցություն", default option "Բոլոր կուսակցությունները"
- Level Select: label "Մակարդակ", default option "Բոլոր մակարդակները", options: Ազգային / Տեղական / Կուսակցական
- On change: update `?party=` / `?level=` query params AND reset `?page=1`
- Both selects rendered using shadcn Select component (not native `<select>`)
- Filter row: `flex flex-wrap gap-4 mb-6`

### Pagination

- State stored in URL query param `?page=N` via `useSearchParams` (react-router-dom)
- Renders: Prev | 1 | 2 | 3 | … | N | Next
- Prev disabled (aria-disabled) when page=1; Next disabled when page=totalPages
- Ellipsis shown when total pages > 7; show first 3, ellipsis, last 3
- Pagination component centered: `flex justify-center mt-6`

### Navigation Active State

- Current page's nav link: `font-semibold text-zinc-900` (zinc-900 text, no background)
- Other links: `text-zinc-600 hover:text-zinc-900` transition-colors
- Active detection: use `NavLink` from react-router-dom with `className` callback

### Promise Stub Click

- Entire PromiseStub row is clickable: `<Link to={/promises/${slug}}>`
- Links to `/promises/{slug}` — resolves to NotFoundPage in Phase 2 (intentional, per D-07)
- Hover: `bg-zinc-50` on the row
- Cursor: pointer

### Avatar / Photo Fallback

- If `photo_url` is non-null: `<img src={photo_url} alt={name_hy} className="rounded-full object-cover" />`
- If null: render zinc-200 bg circle with initials in zinc-700 semibold
- Initials: first char of first word + first char of last word from `name_hy` (Unicode-safe `split(/\s+/)`)
- Grid card size: 64×64px (`w-16 h-16`)
- Profile hero size: 96×96px (`w-24 h-24`)

### Loading States

- All data-fetching pages show skeleton placeholders while `isLoading === true`
- Skeleton: `animate-pulse` bg-zinc-200 rounded blocks matching the shape of the content
- PersonsPage: 6 skeleton cards (2 rows of 3)
- PoliticianProfilePage: avatar skeleton circle + 3 text line skeletons + 5 promise stub skeletons
- ElectionsListPage: 5 skeleton election cards

### Error States

- When `isError === true` on any page: show inline error message (not full-page error)
- Error message: zinc-50 bg, red-600 text, icon `AlertCircle` (lucide), 14px
- See Copywriting section for exact copy
