## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → GET /api/elections | Slug and page params are untrusted |
| FastAPI → PostgreSQL | Correlated subquery uses ORM — parameterized; no string formatting |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-12 | Tampering | GET /api/elections/{slug}/promises path param | mitigate | SQLAlchemy ORM: select(Election).where(Election.slug == slug) — parameterized; no string interpolation |
| T-02-13 | Information Disclosure | GET /api/elections/{slug}/promises returns Promise data | mitigate | Join with PromiseElectionLink AND filter Promise.moderation_status == ModerationStatus.approved |
| T-02-14 | Denial of Service | GET /api/elections with large page | accept | Offset pagination with le=100 cap on per_page; acceptable for Phase 2 read-only |
| T-02-15 | Information Disclosure | ElectionWithCountOut exposes promise_count | accept | Promise count is an aggregate integer; reveals no individual promise content; public by design |
</threat_model>
