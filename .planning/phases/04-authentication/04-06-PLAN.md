    Complete the following setup steps:

    **Google OAuth setup:**
    1. Go to https://console.cloud.google.com/
    2. Create a new project (or use existing) → Credentials → Create Credentials → OAuth 2.0 Client ID
    3. Application type: Web application
    4. Name: Khostumner (dev)
    5. Authorized redirect URIs: add http://localhost:8000/api/auth/google/callback
    6. Copy the Client ID and Client Secret
    7. Go to OAuth consent screen → Add test user: your email address

    **Facebook OAuth setup:**
    1. Go to https://developers.facebook.com/
    2. Create new app → Consumer type
    3. Add "Facebook Login" product
    4. Valid OAuth redirect URIs: add http://localhost:8000/api/auth/facebook/callback
    5. Copy the App ID and App Secret from Settings → Basic

    **Wire credentials:**
    Add to docker-compose.yml backend service environment block:
      GOOGLE_CLIENT_ID: your-actual-google-client-id

      FACEBOOK_CLIENT_ID: your-actual-facebook-app-id
      FACEBOOK_CLIENT_SECRET: your-actual-facebook-app-secret

    **Test OAuth authorize:**
    1. docker compose up backend
    2. Visit http://localhost:8000/api/auth/google/authorize
