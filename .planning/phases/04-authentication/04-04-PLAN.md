## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| RequireAuth → /login redirect | Any unauthenticated navigation is redirected; no client-side secret exposed |
| AuthContext → /api/users/me | Cookie sent automatically; React never reads token value |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-17 | Information Disclosure | AuthContext reads /api/users/me | mitigate | httpOnly cookie — browser sends automatically; JavaScript cannot access token value |
| T-04-18 | Tampering | CSRF token extraction for logout | mitigate | csrftoken cookie is readable (not httpOnly); JS reads it from document.cookie and sends as x-csrftoken header |
| T-04-19 | Elevation of Privilege | RequireAuth flash before redirect | mitigate | Loading spinner shown while isLoading=true prevents flash of protected content before redirect decision |
</threat_model>
